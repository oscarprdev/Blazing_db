## Description

Connect your database, visualize data, and manage it seamlessly using AI-generated queries. No expertise requiredâ€”just simple, intuitive control over your database in one powerful app.

## Stack

- Frontend
  - Typescript
  - React
  - Nextjs
  - Tailwind
  - Tanstack query
  - React flow
- Backend
  - Cloudflare workers   
  - PostgreSQL
  - Cloudlfare R2

## Features
  - Connect to postgreSQL database and visualize tables
  - Ask about your tables with AI
  - Generate and apply generated queries to database
  - Visualize tables fields
  - Store generated queries
